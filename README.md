# 基于微信sdk封装
## 使用
### 安装
* npm
```
npm install --save hndt-wechat
import WeChat from 'hndt-wechat'
```

* script
```
<script src='https://cmsres.dianzhenkeji.com/source/js/hndt-wechat.js'></script>
```
### 初始化
```new WeChat(appId,{title,desc,imgUrl})```

### 参数
* appId 必填，可填空字符串
* title:分享标题
* desc:分享描述
* imgUrl:分享图片
